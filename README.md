### How To run?<br>
まずWindows10 OSのはいったパソコンを用意する<br>
次にPython環境を構築する<br>
＊"Path"も通しておく<br>
最後にcmdで
python run.py
を実行する<br>
First, prepare a computer with the Windows 10 operating system.<br>
Next, create a Python environment.<br>
＊Make sure you have a "Path"<br>
Finally, run cmd
python run.py
with cmd.<br>
